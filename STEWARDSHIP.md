# Stewardship

The Apache License 2.0 defines the legal permissions and obligations for Pocok. Nothing in this document adds a condition or narrows those permissions.

If Pocok saves meaningful work, please consider:

- retaining visible project attribution when practical;
- reporting defects with a focused reproduction;
- contributing generally useful fixes upstream;
- sharing how the package is used so its contracts can improve;
- contacting the maintainers before presenting a substantially unchanged fork as the original project.

Students, teachers, hobbyists, researchers, companies, and commercial applications are all welcome under the same license.
