# Agent prompts

`agent-base.prompt.md` and `agent-base.plan.md` are compatibility entry points for repository agents.

- Use the execution prompt for ordinary repository work.
- Use the plan guide only when a durable plan is justified by risk, duration, or handoff.
- Topic-specific plans remain passive evidence and must be reconciled with current source before execution.

Stable Pocok policy lives in root `AGENTS.md`. General collaboration behavior lives in `docs/agentic-workflow.md`. Current temporary state lives in `docs/current-handoff.md` and qualifying session records.
