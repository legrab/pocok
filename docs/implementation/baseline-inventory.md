# Imported baseline inventory

The imported baseline contained four active packages:

- `Pocok.Primitives`
- `Pocok.Conversion.Abstractions`
- `Pocok.Conversion`
- `Pocok.Hosting`

The baseline also contained package samples, unit tests, architecture tests, packaging tests, duplicated package-specific publication workflows, an external-consumer smoke harness, and a package-content audit.

The consolidation deliberately treats existing behavior as migration input. Package boundaries are not preserved merely because they existed in the extracted source.
